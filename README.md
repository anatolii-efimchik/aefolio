# aefolio GitHub Pages

Статический сайт портфолио для GitHub Pages.

## Как опубликовать

1. Создайте новый репозиторий на GitHub.
2. Загрузите в него содержимое этой папки: `index.html`, `styles.css`, `README.md`.
3. В настройках репозитория откройте `Pages`.
4. Source: `Deploy from a branch`.
5. Branch: `main`, folder: `/root`.
6. Сохраните настройки.

Через несколько минут сайт будет доступен по адресу GitHub Pages.

## Локальный просмотр

Откройте `index.html` напрямую в браузере через `file:///`.
